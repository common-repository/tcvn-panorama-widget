A simple widget to display 360 degree images in a position.

== Description ==

The TCVN Panorama Widget for Wordpress is a simple module to display 360 degree images in module positions. It works with jQuery, neither flash nor any other player is required. The overall width and height can be customized and you can setting effect eg: speed, direction, start position and more. And this module supports many image type: .jpg, .png and .gif and easy to custom layout.

Features:

- Supports .jpg, .png and .gif
- Choose if autostart scrolls left or right.
- Very easy to customize image width, height and setting effect.
- Customize User CSS formatting.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Tableless design and 100% CSS-based
- Fully compatible IE7+, Firefox 2+, Flock 0.7+, Netscape, Safari, Opera 9.5
- Easy installation

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire 'tcvn-panorama-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'TCVN Panorama Widget' to your sidebar
